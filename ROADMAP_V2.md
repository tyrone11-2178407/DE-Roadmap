# Roadmap V2 — Foundation-First, 3 Tracks

> **Top 3 tracks:** Big 4 Tech Consulting · Tech Sales New-Grad Programs · Product/Data Analyst at tech
> **Application peak:** October 1, 2026
> **Offer target:** December 2026
> **Comp aim:** $80–110k Y1 → $150–200k+ by Y3–Y5

## Strategy in 1 paragraph

Build the **shared foundation** first (SQL + Excel + Dashboards + STAR + Resume + Portfolio). All 3 tracks need this same base. Once foundation is solid, do **2–3 weeks of track-specific prep** per track. Apply broadly, deep-prep top 2 of 3.

## Why foundation-first works for you

- ADHD brain works better with **one focus at a time**, not split prep
- Skills compound — better SQL helps in Big 4 cases AND analyst screens AND sales product demos
- Anxiety is lower when you build proof of skill before applying
- If you discover a track doesn't fit during branch phase, you don't lose foundation work

## Phase 1: FOUNDATION (10 weeks · May 7 – Jul 16)

Everything every track needs. No track-specific work yet.

### Week 1–4: SQL Mastery

**Concepts**
- JOINs (inner/left/right/full/self)
- Window functions (ROW_NUMBER, RANK, LAG/LEAD, PARTITION BY)
- CTEs and subqueries
- Aggregations + GROUP BY edge cases
- Date/time manipulation
- Query optimization basics

**Resources:** StrataScratch (paid ~$30) · Mode SQL Tutorial (free) · DataCamp SQL tracks · select-star.io · LeetCode SQL section

**MVP:** 25 medium SQL problems solved, can write a window function from memory.
**Done:** 75 problems solved, 4-table JOIN with window function in <10 min.
**Deliverable:** SQL drill log (75+ problems linked).

### Week 5–7: Excel + Tableau or Power BI

**Concepts**
- Excel: VLOOKUP, INDEX/MATCH, XLOOKUP, pivot tables, SUMIFS, conditional formatting
- Tableau OR Power BI (pick ONE): connect data, 3 chart types, calculated fields, filters, full dashboard
- Data storytelling: titling, highlighting, removing noise

**Resources:** Tableau Public free training · Power BI Microsoft Learn · ExcelJet (free) · Optional cert: Tableau Specialist ($100) or PL-300 ($165)

**MVP:** 1 dashboard live on Tableau Public from a Kaggle dataset.
**Done:** 3 dashboards live, can story-tell any chart in 30 sec.
**Deliverable:** Public Tableau Public profile with 3+ dashboards.

### Week 8–10: Just-Enough Python

**Concepts**
- pandas: read_csv, filter, groupby, merge, pivot, sort
- Basic: loops, functions, list/dict comprehensions
- 1 small visualization (matplotlib basics)
- **Skip:** classes, decorators, async, ML libraries

**Resources:** DataCamp pandas track · Real Python pandas articles · Kaggle Learn (free)

**MVP:** Write a function with for-loop + pandas groupby without AI.
**Done:** CSV → clean → analyze → visualize in 30 min. Comfortable in pandas interview talk.
**Deliverable:** 1 mini analysis notebook on GitHub.

### Throughout Phase 1 (parallel mini-tracks)

- **STAR stories:** draft 1–2/week → 10 stories by end of Phase 1
- **Resume polish:** initial draft Week 1, refine Week 5, NWU career services review Week 8
- **LinkedIn:** update photo, headline, About, projects, skills
- **LeetCode side track:** 30 min/day (NeetCode 150 — Easy first)

**Phase 1 success metric:** SQL drill log + 3 dashboards + 1 pandas notebook + 10 STAR stories + polished resume + updated LinkedIn.

## Phase 2: PORTFOLIO PROJECT (3 weeks · Jul 16 – Aug 6)

One end-to-end project that proves all foundation skills.

**Project spec:**
- Pick a dataset you actually care about (sports, finance, music, gaming, your school data)
- SQL queries to answer 3–5 business questions
- 1 dashboard (Tableau or Power BI) telling the story
- 1 pandas notebook with deeper analysis
- 1-page write-up: problem, approach, finding, recommendation
- Public on GitHub + Tableau Public

**Why this matters:** Every interview asks "show me something you've built." Most candidates have nothing or ChatGPT-generated junk. You have a real project you can defend.

**MVP:** Project live with at least the dashboard + write-up.
**Done:** SQL + dashboard + pandas notebook + write-up all linked from a clean GitHub README.
**Deliverable:** Portfolio repo on GitHub.

## Phase 3: TRACK BRANCHES (4 weeks · Aug 6 – Sep 3)

Now the tracks split. Deep-prep **2 of 3 branches**. Apply to all 3 anyway.

### Branch A — Big 4 Tech Consulting

**Focus:** light case prep + cover letter tailoring + behavioral

- Read **Case Interview Secrets** (Victor Cheng, free PDF) — 1 day
- Drill 10–15 cases (NOT 50 — Big 4 cases are way lighter than MBB)
  - Market sizing × 4
  - Profitability × 4
  - Tech-flavored business problems × 4
- 5 STAR stories tailored to consulting LPs (analytical, leadership, ambiguity)
- Cover letter template per firm × 4 firms
- Mock cases on Pramp or NWU career services × 5

**Done:** Can run a 30-min case end-to-end. Cover letters drafted for all 4 firms.

### Branch B — Tech Sales New-Grad Programs

**Focus:** pitch + role-play + business curiosity

- Watch 5 SDR/AE day-in-the-life videos
- Practice "Tell me about yourself" → 90 sec, 3 versions
- Practice "Why sales?" → tied to your real story
- Cold call role-play × 10 (Pramp or with a friend)
- 1 mock SDR/BDR final round (NWU career services)
- Read 1 sales book: SPIN Selling OR The Challenger Sale (library)
- Research 5 target SaaS products → why they exist, who they sell to, why you'd love selling them

**Done:** Can pitch any product to a stranger. 5 program-specific cover letters.

### Branch C — Product/Data Analyst

**Focus:** SQL portfolio depth + product sense + A/B test reasoning

- Add 2 more dashboards to portfolio (1 must be a "user funnel" analysis)
- Read **Trustworthy Online Controlled Experiments** (Kohavi — library) OR free Eric Colson articles on A/B testing
- Practice product sense questions: "How would you improve [Spotify, Notion, Asana]?"
- 5 product analytics interview questions on InterviewQuery
- 1 mock product analytics interview

**Done:** Portfolio has 5+ dashboards. Can answer "how would you measure success of X feature?" structurally.

### Throughout Phase 3 (continued)

- LeetCode side track continues (now in mediums)
- Update Career HQ application tracker (build the spreadsheet/app empty so it's ready)

## ADHD Mechanics (kept from V1, all still active)

- **Grace days** — 2 free skips/month, auto-applied
- **Floating calendar** — durations not deadlines, only Oct 1 application peak is fixed
- **Minimum Viable Day (MVD)** — 5-min contact still counts (1 SQL problem read, 1 article paragraph, 1 mock answer drafted)
- **Buffer weeks** — 4 weeks built between Phase 3 end (Sep 3) and Oct 1
- **Ship MVP buttons** — 30%-done → declare phase complete and move on
- **Track Lock** — locked on these 3 tracks through Aug 1. Re-evaluate requires 200+ char written reasoning + Switching Cost calc.

## Decision Tree (Kill Switches)

- **By June 1, SQL not at MVP** → drop Phase 1 Python, double down on SQL
- **By July 30, Phase 1 not done** → skip Phase 2 portfolio project, go straight to Phase 3 branches with what you have
- **By Sep 3, no portfolio** → drop LeetCode entirely, ship anything
- **By Oct 15, no first-round interviews** → drop Track 1 (Big 4) heavy-prep, lean Track 2 (Sales) and Track 3 (Analyst safety net)
- **By Dec 1, no offers** → extend timeline 3 months · broaden Track 3 to government + non-profit analyst roles
